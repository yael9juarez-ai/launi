# **App Name**: UniEats

## Core Features:

- Secure University Login: Client and staff authentication via institutional email, managing user roles and session integrity for a personalized experience.
- Dynamic Menu Browsing: Customers can view a categorized cafeteria menu complete with item images, prices, and detailed descriptions on a responsive interface.
- Streamlined Online Ordering: Enables customers to place pre-orders, select specific pick-up times, and customize items for efficient and quick service.
- Personalized Order History: Allows customers to review their past purchases and current order statuses, with an option for quick reordering of frequently chosen items.
- Real-time Order Dashboard: A dashboard for cafeteria staff to manage incoming orders in real-time, update their status (e.g., preparing, ready for pickup), and monitor fulfillment progress.
- Admin Menu Management: An administrative interface for cafeteria managers to create, edit, remove menu items, adjust prices, and upload high-quality product images to the database.
- Smart Menu Recommendation Tool: A generative AI tool that suggests popular menu items or new dishes to clients, reasoning on historical sales data and real-time inventory or seasonal availability to inform its recommendations.

## Style Guidelines:

- Primary color: A vibrant institutional red, representative of UNI's identity, for key interactive elements like buttons and highlights (#E30613).
- Background color: A very light, almost white tone with a subtle warmth, providing a clean and elegant backdrop for content (#F7F5F5).
- Accent color: A deep, rich charcoal grey with a hint of red, used for primary text, darker interface elements, and subtle contrast (#2A2626).
- Main typeface: 'Inter', a clear and modern sans-serif, suitable for both headlines and body text to ensure readability and institutional aesthetic across all content.
- Iconography should be simple, professional, and easily understandable, reinforcing the clean and modern institutional aesthetic.
- The layout is designed to be modern, clean, and highly institutional, ensuring optimal viewing and interaction across desktops, tablets, and mobile devices.
